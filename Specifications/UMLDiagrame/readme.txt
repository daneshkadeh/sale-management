=========================================== Hướng dẫn viết tài liệu UML =======================================================
- Các tài liệu UML (database model, use-case, activity diagram) được viết bởi phần mềm StarUML.
 + ThuSports_ER_diagrame_EN.vsd: sử dụng phần mềm MS Visio, vẽ tất cả các table trong project SSM (model cũ dùng để tham khảo, hiện tại sử dụng ThuSportModel.uml)
 + SSM_Vietnamese_English: Bảng dịch Anh-Việt các thuật ngữ (để hiểu rõ tên của các table).
 + ThuSportModel.uml: vẽ tất cả các table trong project SSM, cấu trúc module
 + ssm_config.uml: vẽ tất cả các table trong module Config.
 + ssm_catalog.uml: vẽ tất cả các table trong module Catalog.
