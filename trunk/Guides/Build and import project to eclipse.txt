1. Set up enviroment:
	Refix the paths in file set_env.bat (%PathToProject%\SaleManagement\ssm\tools\set_env.bat)
	Run it in the cmd
2. Build the project by maven the command:
	"mvn clean install"
3. To make the project can be imported to eclipse, run the command:
	"mvn clean eclipse:eclipse"
	Or "mvn clean eclipse:eclipse -DdownloadSources=true -DdownloadJavadocs=true" if you want to download source, javadoc.
4. 
5. Import to eclipse
